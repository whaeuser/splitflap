Splitflap Color Test

Anleitung:

  1. Öffne flipboard.html in einem neuen Tab
  2. Öffne die Browser Console (F12)
  3. Führe diese Befehle aus:

Test-Befehle:

Test 1: Einzelne Farbe

window.splitflapAPI.setDisplay('', 'GRUEN TEST', '', '', '', '', null, 'gruen');

Test 2: Mehrere Farben

window.splitflapAPI.setDisplay('WEISS', 'GRUEN', 'ROT', 'BLAU', 'GELB', 'ORANGE', 'weiss', 'gruen', 'rot', 'blau', 'gelb', 'orange');

Test 3: Farbe auf bestehende Zeile setzen

window.splitflapAPI.setLineColor(1, 'rot');

Test 4: Prüfe aktuelles currentColors Array

console.log('Current colors:', currentColors);

Test 5: Prüfe COLORS Objekt

console.log('COLORS:', COLORS);

Test 6: Manuell Farbe setzen

document.getElementById('line2').querySelectorAll('.flip-char-content').forEach(el => {
    el.style.color = '#2ECC71'; // Grün
    console.log('Color set to:', el.style.color);
});
